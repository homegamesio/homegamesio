Homegames is an open source LAN-based game system built with JavaScript.

Homegames is intended to be a fun way to build games and play them with your friends without requiring a ton of prior programming experience. It is currently in active development by a small team as a side project.

Getting Started

To get started, check out the project on GitHub. At the moment, the project is still under active development and isn't really ready for people to use yet. If you're curious about how it works or just want to start experimenting with potentially broken code, feel free to check out the repo and start hacking away.
