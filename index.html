Homegames is an open source game platform built with JavaScript.

The project at this point is very much a work in progress. Our ultimate goal is for Homegames to be a platform where people can play games built by the Homegames community or build their own without needing any significant programming experience.

Homegames is and always will be available for free. At some point in the future we may offer a premium hosted version, but we don't currently have any solid plans to do so.

We've been working on Homegames for a little over a year. We currently have our core game server and a web app, and we're planning on building iOS & Android apps at some point within the next year. The project is actively maintained by Joseph Garcia & Yazeed Loonat.

Getting Started

To get started, check out our wiki on GitHub. At the moment, the project is still under active development and things change pretty frequently. If you're curious about how it works or just want to start experimenting, feel free to check out the repo and start hacking away.

We occasionally update picodeg.io with a running instance of Homegames that can be accessed over the public Internet. Sometimes it breaks. If it does, feel free to complain to us on Twitter.

Downloads

We've created executable versions of Homegames for MacOS, Windows & Linux using pkg. Ideally, these would be versioned snapshots taken automatically every time a change is made to the master branch, but we're not quite there yet. At the moment, we update these manually so the downloads available here might not be the most up-to-date version of the project.

To run Homegames, you simply need to download and run the game & app servers, then navigate to homegames.link in any browser on the same network as the servers. You will likely have to tell Windows or MacOS to allow running software from unidentified developers in order to actually run the servers. You can watch our getting started video to see how this is done. If something is broken, please let us know by using the contact info at the bottom of the page.
