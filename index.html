Homegames is an open source LAN-based game system built with JavaScript.

Homegames is intended to be a fun way to build games and play them with your friends without requiring a ton of prior programming experience. It is currently in active development by a small team as a side project.

We occasionally update Picodegio with a running instance of Homegames that can be accessed over the public Internet. Sometimes it breaks.

Getting Started

To get started, check out our wiki on GitHub. At the moment, the project is still under active development and things change pretty frequently. If you're curious about how it works or just want to start experimenting, feel free to check out the repo and start hacking away.

Downloads

We've created executable versions of Homegames for MacOS, Windows & Linux using pkg. Ideally, these are versioned snapshots taken automatically every time a change is made to the master branch, but we're not there yet. These executables were created from this commit.

To run Homegames, it should be as easy as running both the game server & app server, then navigating to homegames.link in any browser on the same network as the server. You will likely have to tell Windows or MacOS to allow running software from unidentified developers in order to actually run the servers. If something is broken, please let us know by using the contact info at the bottom of the page.
